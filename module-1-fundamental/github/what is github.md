# What is GitHub?

GitHub is a product of Microsoft.

## Purpose of GitHub
- GitHub is used to create repositories (server data upload).
- GitHub is used to share data via links.
- GitHub allows sharing data in public or private.

## Steps to Use GitHub
1. **Create an account on GitHub:**  
   [Sign up here](https://github.com/)
2. **Sign in to GitHub.**
3. **Create a new repository:**  
   [Create repository](https://github.com/Brijesh1990/data_analytics_5-30to7-00.git)
4. **Download Git software for data upload:**  
   [Download Git](https://git-scm.com/install/windows)
5. **GitHub commands:**
   - `git init`
   - `git add .`
   - `git commit -m "20-01-2026 all data uploaded"`
   - `git remote add origin https://github.com/Brijesh1990/data_analytics_5-30to7-00.git`
   - `git push -u origin master`
